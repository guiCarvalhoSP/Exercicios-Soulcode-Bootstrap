Beneficios de usar o bootstrap:

Horas poupadas estilizando: Pessoas satifeitas: Quantidade de classes

6H em média(Combinado com Jquery) 33.234 Dados ficticios ilustrativos

  • As listas no bootstrap podem ser feitas utilizando a classe .list-group, que cria um grupo para uma lista, e que não precisa necessariametne ser uma lista do html

  • Pode se usar até links ao invés de listas, bastando colocar os links com a classe .list-group-item, dentro de uma div com a classe .list-group, para criar uma lista.

  • As listas podem ser estilizadas de diversas maneiras, alterar cor, formato, colocar badges e etc.

  • Pode se usar a classe .flush para tirar bordas e arredondamentos, ou utilizar as classes .list-group-item-* para alterar cor. As possibilidades são diversas

  • Listas são uma mão na roda para apresentar conteúdo de forma organizada.

  • E utilizando o bootstrap nunca foi tão fácil fazer e estilizar elas.
